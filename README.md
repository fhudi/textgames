# TextGames

## Setup
```
❱❱❱ pip install -r requirements.txt
```

## Play (Terminal)
```
❱❱❱ python play.py
```

## Play (Web UI)
```
❱❱❱ pip install gradio
❱❱❱ GRADIO_SERVER_PORT=1080  python play_gradio.py
```
Open `localhost:1080` to access.

## Optional Environment Varibles
```
TEXTGAMES_SHOW_HIDDEN_LEVEL=1
```
